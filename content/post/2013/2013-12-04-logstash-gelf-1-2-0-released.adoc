I've released today logstash-gelf 1.2.0, I'll be available until this evening on Maven Central.

<!--more-->

This release contains following changes:

* Support for log4j2 (2.0-beta9)
* Changes to property names for host/port (before graylogHost/graylogPort, now host/port)
* Adjusted documentation to reflect the changes
