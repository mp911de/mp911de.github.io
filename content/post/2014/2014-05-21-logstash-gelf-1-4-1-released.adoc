I've released today logstash-gelf 1.4.1, I'll be available until this evening on Maven Central.

<!--more-->

This release contains following changes:

* Logback appender should use formatted message
* JBoss Wildfly 8.0.0.Final startup failure with property "threshold" not defined/Documentation fixed
* Redis bugfix and GelfSender flexibility

Thanks to felixhamel, plozi and Batigoal
