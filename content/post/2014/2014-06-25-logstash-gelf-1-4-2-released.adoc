I've released today logstash-gelf 1.4.2, I'll be available until this evening on Maven Central.

<!--more-->

This release contains following changes:

== Fixes

* https://github.com/mp911de/logstash-gelf/issues/21[++#++21] Redis and commons-pool2 dependencies needed when not using redis. This bug is severe and caused NoClassDef-Errors so you can't use logstash-gelf without jedis/commons-pool2 libraries. If you don't use redid and you want a small library set it's recommended to update to version 1.4.2.

== Enhancements

* https://github.com/mp911de/logstash-gelf/issues/20[++#++20] Configuration of default field names enhancement
* https://github.com/mp911de/logstash-gelf/issues/19[++#++19] Code Cleanup enhancement
