I've released today logstash-gelf 1.5.1. This is a bugfix release for JBoss AS7/Wildfly users who use the NDC/MDC.

<!--more-->

This release contains following changes:

== Fixes

* https://github.com/mp911de/logstash-gelf/pull/29[++#++29] JBoss 7 Logging Modification for Async support (thanks to https://github.com/Batigoal[@Batigoal])
