I've released today logstash-gelf 1.5.3. This is a bugfix release for Hadoop Hive users. I did not announce release 1.5.2 the last time. Apologies for that.

<!--more-->

This release contains following changes:

== Fixes

* https://github.com/mp911de/logstash-gelf/issues/30[++#++30] Passing originHost from log4j2.xml (1.5.2)
* https://github.com/mp911de/logstash-gelf/issues/31[++#++31] "No default-logstash-fields.properties resource present" output breaks Apache Hive (1.5.3)
* Polished documentation for stack trace filter
