I released today logstash-gelf 1.5.4. This release carries some internal improvements if you plan to extend log appenders and GELF senders.

<!--more-->

This release contains following changes:

== Enhancements

* https://github.com/mp911de/logstash-gelf/pull/34[++#++34] Sender specific configurations on sender factory (thanks to https://github.com/Batigoal[@Batigoal])
