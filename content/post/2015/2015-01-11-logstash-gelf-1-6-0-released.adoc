I released today logstash-gelf 1.6.0. This release contains support for GELF 1.1 spec, Redis Sentinel and the Datenpumpe.

<!--more-->

This release contains following changes:

== Fixes

* https://github.com/mp911de/logstash-gelf/issues/39[++#++39] Check for additionalFields null value fixes

== Enhancements

* https://github.com/mp911de/logstash-gelf/issues/33[++#++33] Increase dependency versions
* https://github.com/mp911de/logstash-gelf/issues/35[++#++35] Support GELF 1.1 spec
* https://github.com/mp911de/logstash-gelf/issues/36[++#++36] Add Redis Sentinel support
* https://github.com/mp911de/logstash-gelf/issues/37[++#++37] Fix NPE when no log4j MDC available
* Documentation about Sentinel and Datenpumpe
