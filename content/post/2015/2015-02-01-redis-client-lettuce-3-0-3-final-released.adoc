I released lettuce 3.0.3. This is a bugfix release fixing a memory leak in pubsub and adds stability when reconnecting.

<!--more-->

This release contains:

This release fixes a memory leak when using PubSub.

== Fixes

Memory leak on connection loss https://github.com/mp911de/lettuce/issues/26[++#++26]

Fix channel loss on reconnect https://github.com/mp911de/lettuce/pull/27[++#++27] (Thanks to https://github.com/kichik[@kichik])

Resubscribe even when there are no listeners https://github.com/mp911de/lettuce/pull/30[++#++30] (Thanks to https://github.com/kichik[@kichik])

== Other

* Allow passing server, port and password on command line for tests https://github.com/mp911de/lettuce/pull/29[++#++29]
