I released the first version of https://github.com/mp911de/spinach[spinach], a scalable Java https://github.com/antirez/disque[Disque] client based on http://netty.io[netty]. spinach is built on top of the https://github.com/mp911de/lettuce[lettuce] client which provides SSL and Unix domain connections in addition to plaintext connections.

== *Basic* usage

[source,java]
----
DisqueClient client = new DisqueClient("host");
DisqueConnection<String, String> connection = client.connect().sync();
DisqueCommands<String, String> sync = connection.sync();
String jobId = sync.addjob("queue", "body", 1, TimeUnit.MINUTES);

Job<String, String> job = sync.getjob("queue");
connection.ackjob(job.getId());
----

== *Features*

* Works with Java 6, 7 and 8
* https://github.com/mp911de/spinach/wiki/Basic-usage[synchronous] and https://github.com/mp911de/spinach/wiki/Asynchronous-Connections[asynchronous] APIs
* https://github.com/mp911de/spinach/wiki/SSL-Connections[SSL] and https://github.com/mp911de/spinach/wiki/Unix-Domain-Sockets[Unix Domain Socket] connections
* https://github.com/mp911de/lettuce/wiki/Codecs[Codecs] (for UTF8/bit/JSON etc. representation of your data)
