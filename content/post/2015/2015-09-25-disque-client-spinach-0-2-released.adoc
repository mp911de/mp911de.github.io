I released https://github.com/mp911de/spinach[spinach] 0.2 today. image:/images/external/github-emoji/1f422.png[/images/external/github-emoji/1f422]image:/images/external/github-emoji/1f422.png[/images/external/github-emoji/1f422]

This release addresses minor issues and allows to use lettuce 3.3.Final. It adds support for more Disque commands and eases creation of a DisqueURI.

<!--more-->

== Enhancements

* Provide a Reactive API https://github.com/mp911de/spinach/issues/4[++#++4] (shipped with 0.1.1)
* Implement CLUSTER commands https://github.com/mp911de/spinach/issues/5[++#++5] (shipped with 0.1.1)
* Implement JSCAN command https://github.com/mp911de/spinach/issues/6[++#++6] (shipped with 0.1.1)
* Implement NACK command https://github.com/mp911de/spinach/issues/11[++#++11]
* Upgraded lettuce from 3.2.Final to 3.3.Final https://github.com/mp911de/spinach/pull/13[++#++13] (Thanks to https://github.com/pulse00[@pulse00])

== Other Changes

* Bump rxjava to 1.0.14
* Polishing and sugar for DisqueURI

For complete information on spinach see the websites:

* https://github.com/mp911de/spinach[http://github.com/mp911de/spinach]
* http://spinach.paluch.biz.
